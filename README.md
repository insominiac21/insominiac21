## 🚀 Featured Projects

### 🔹 Sahayak AI: Multilingual WhatsApp Voice RAG Assistant
`WhatsApp • Twilio • Sarvam AI • Qdrant • Supabase • Render`
- Built voice-enabled WhatsApp assistant for govt schemes in 22 Indian languages.
- Designed pipeline: STT → translate → retrieve → generate → translate back.
- Added semantic search, metadata filtering, and citation-style grounding.

### 🔹 NEM AI (Viva-AI): RAG-Powered Study Mentor
`RAG • Prompt Engineering • Guardrails • Adaptive Scheduling`
- Built multi-turn viva system adapting difficulty in real-time.
- Added prompt-injection protection, schema validation, and spaced-repetition scheduling.
- Designed structured question generation using hint-first scaffolding.

### 🔹 Real-Time Shoplifting Detection using YOLOv8
`OpenCV • Object Detection • Flask • MJPEG Streaming`
- Created real-time retail surveillance for live anomaly detection.
- Achieved stable performance via temporal validation and confidence thresholding.
- Supported webcam, DroidCam, and IP/RTSP via camera-agnostic ingestion.

### 🔹 UIDAI Aadhaar Enrollment Operational Intelligence
`Big Data • Geospatial Analysis • Clustering • Dashboard`
- Processed 5.4M+ Aadhaar records across 1,045 districts.
- Built district embeddings yielding clustering-based operational archetypes.
- Developed priority scoring framework and similarity-based district retriever.

### 🔹 Hybrid Ranking & Recommendation Engine
`Recommender Systems • Ranking • Feature Engineering`
- Built hybrid recommendation pipeline utilizing collaborative filtering and content signals.
- Improved top-N ranking (Precision@K, Recall@K, NDCG, MAP).
- Optimized sparse-matrix inference, lowering latency and memory usage.

### 🔹 Coral Reef Degradation Forecasting
`Forecasting • EDA • Ensemble Modeling`
- Analyzed CREMP environmental datasets studying coral health decline.
- Compared regression pipelines, utilizing stacking ensembles for predictive stability.
- Identified environmental drivers of reef degradation via feature analysis.
